# Blog Server API

- Server API for an exercise from [The Odin Project](https://www.theodinproject.com/) that recreates a simple fake blog.

## Tecnologies

- NodeJS
- ExpressJS
- MongoDB
- MongooseJS
- PassportJS
- JsonWebToken
- BcryptJS
- Slugify
